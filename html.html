반응형 웹 vs 적응형 웹

현대인들의 필수품은 바로 스마트폰!
노트북, 데스크톱 등 다양한 기기가 존재하지만 웹 사이트 방문율이 가장 높은 것은 모바일 기기이다.
따라서 모든 기기와 플랫폼 에서 원활히 작동되며 디자인 면에서도 뛰어난 웹 사이트가 트렌드가 되며 떠오른 디자인

반응형 웹디자인
감지된 화면 크기에 따라 자동으로
페이지가 재배열되는 유동적인 접근 방식
CSS를 사용하여 최종 사용자가 사용하는
기기의 특성을 검사한 후 자체적으로 렌더링
적응형 웹 디자인
브라우저가 주어진 플랫폼에 맞춰
특별히 생성된 레이아웃을 불러오는 방식
웹 페이지에서 감지된 기기를 기반으로
미리 만들어진 정적인 레이아웃 불러옴
